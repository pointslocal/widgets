Gameplan Widgets Gallery

Gameplan's widget library provides responsive, deeply-customizable widgets to meet the needs of any sport. From player cards to scoreboards to brackets to pickems, we've got you covered.

Just a few lines of code and you're up and running. And when it's time to build something truly unique, these widgets are built directly on a robust, flexible REST API, which means there's no limit to what your widgets can do.

Requirements

Gameplan widgets require jQuery and the Gameplan javascript library. To use default styles (a responsive grid), also include the gameplan CSS file

A minimal installation can be seen with the following:

<script src="https://ajax.googleapis.com/ajax/libs/jquery/2.2.4/jquery.min.js"></script>
<script src="//cdn.pointslocal.com/js/pointslocal.widgets.js"></script>
<link href='//cdn.pointslocal.com/css/pointslocal.widgets.css' rel='stylesheet' type='text/css'>

Each widget is constructed with a new Pointstlocal() call. The first parameter is a DOM selector and the second are options passed into the widget. In addition to standard options, any additional passed to the widget will be proxied along to the resulting API call, allowing for total flexibility of content requested.

Upcoming / Scoreboard

The upcoming/scoreboard widget will show most recent scores or next upcoming games if no recent scores exist. It updates automatically as games are updated in the admin.

Dates and times can be formatted using the dateFormat and timeFormat parameter

      var g1 = new Pointslocal('#gameplan-scoreboard-mini', {
        type: 'games',
        site:'miami-fl',
        count: 4,
        dateFormat: 'M j',
        template: 'gameplan.scoreboard'
      });

Standings

The standings widget allows you to specify sport and class and show the records and standings. Note this requires comprehensive coverage as it calculates standings based on games within a season.

      var g2 = new Pointslocal('#gameplan-standings', {
        type: 'standings',
        site:'miami-fl',
        count: 1,
        dateFormat: 'M j',
        template: 'gameplan.standings'
      });

Team Cards

Team cards show a snippet of information and a logo from a team with last result (or next upcoming). 

      var teamWidget = new Pointslocal('#gameplan-teamcards', {
        type: 'teams',
        site:'omaha',
        school: 'adams-central',
        count: 1,
        template: 'gameplan.team'
      });
